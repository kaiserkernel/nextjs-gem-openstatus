---
title: What is OpenStatus?
---

OpenStatus is an open-source monitoring service with a beautiful status page and
incident management.

## Trying OpenStatus

The easiest way to quickly try the app is to signup on
[www.openstatus.dev](https://www.openstatus.dev)

The signup is free.

## Getting Started

There are three ways to get started with OpenStatus.

- **Cloud**: the fastest and easiest way to try the app (it's free)
- **Local**: if you're a developer and would like to experiment or contribute to
  the app
- **Self-hosting**: if you want greater control over your data and want to run
  the app on your own server
