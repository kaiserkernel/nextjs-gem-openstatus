We are excited to announce the release of the Terraform provider for OpenStatus.
This provider allows you to manage your Grafana Cloud resources using Terraform,
enabling you to define and manage your OpenStatus resources as code.

Here's the link to the:
https://registry.terraform.io/providers/openstatus/openstatus/latest/docs
